## What is this?

A random number generator

## Installation

Run `npm i sangee_random_number_generator`

Use:

```
import rng from 'sangee_random_number_generator';

rng(5,10)
```

## parameters

The Random Number Generator accepts two paramaters, min and max values.

By default
'min' is set to 0 and
'max' is set to 100
